# Software Requirements

| ID    | Type        | Priority | Description |
|------|-------------|----------|-------------|
| FR-01 | Functional | Must | The system shall allow the operator to create spa services with price and duration. |
| FR-02 | Functional | Must | The system shall allow the operator to create customers with name and species. |
| FR-03 | Functional | Should | The system shall allow the operator to create bookings by assigning a customer to a spa service at a specific date and time. |
| FR-04 | Functional | Must | The system shall calculate total income based on the prices of all booked spa services. |
| FR-05 | Functional | Must | The system shall allow the operator to record expenses and calculate the current profit. |
| FR-06 | Functional | Must | The system shall persist services, customers, bookings, and expenses using JSON files. |
| FR-07 | Functional | Should | The system shall determine and display the most frequently booked spa service. |
